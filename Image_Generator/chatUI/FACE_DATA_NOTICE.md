# Face data attribution and licensing — Chat_Memory_Test

The 12 face avatars used by the **Chat_Memory_Test** benchmark task are
**synthetic AI-generated faces** — they do not depict any real person.

## Source

- **HuggingFace dataset**: [`javi22/this-person-does-not-exist-10k`](https://huggingface.co/datasets/javi22/this-person-does-not-exist-10k)
- **Author**: Javier (HuggingFace user `javi22`)
- **Format**: 10,000 individual JPG files (1024×1024)
- **License**: **MIT License** (commercial + research use, attribution required)

The dataset name and content indicate the images were generated by a
StyleGAN-family model (the same generator class behind
[thispersondoesnotexist.com](https://thispersondoesnotexist.com)). They
are wholly synthetic and contain no real person's identity.

## How we use it

`Image_Generator/chatUI/fetch_faces.py` downloads exactly **12 individual
face images** by fixed indices via `huggingface_hub.hf_hub_download` from
the public HuggingFace dataset. Indices are hard-coded so the selection
is reproducible. We do not redistribute the source dataset itself; users
who run `fetch_faces.py` pull directly from HuggingFace.

The 12 faces are renamed (`P01_marcus.jpg`, `P02_priya.jpg`, ...) and
composited as circular crops into rendered chat-UI screenshots by
`Image_Generator/chatUI/chat_renderer.py`. The rendered screenshots are
stored under `Benchmark_Pipeline/data/image/Chat_Memory_Test/` and are
gitignored — they are regenerated on demand from the build script.

## Why synthetic faces (and why this dataset)

Chat_Memory_Test forces methods to remember which face avatar maps to
which persona across overlapping social networks. Real face photos
would create privacy and consent problems for both the depicted person
and the benchmark distribution. Using a publicly licensed, MIT-licensed
synthetic dataset avoids both concerns:

- **No real identity**: nobody can be re-identified from any of the 12
  faces — they are GAN samples, not photos of real people
- **Permissive license**: MIT lets us redistribute through the script
  pipeline (the script downloads from HF; we don't bundle the originals)
- **Reproducible selection**: fixed indices guarantee anyone running
  `fetch_faces.py` gets the exact same 12 faces

If you need to swap the face source, edit `PERSONAS` and `INDICES` in
`fetch_faces.py`. Any other licensed synthetic-face dataset works as a
drop-in replacement; the renderer only needs JPG/PNG files at
`Benchmark_Pipeline/data/image/Chat_Memory_Test/faces/{persona_id}_{name}.jpg`.

## What this dataset is NOT

- It is **not** real photos of identifiable people
- It is **not** scraped from the open web
- It is **not** subject to GDPR / CCPA personal-data rules (no PII)
- It is **not** redistributed by this repo — `fetch_faces.py` pulls
  directly from HuggingFace at run time
